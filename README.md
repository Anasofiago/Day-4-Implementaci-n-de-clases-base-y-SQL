## Descripción
El proyecto consiste en el desarrollo de una aplicación de escritorio, la cual registre clientes y órdenes.
Dentro de esta realizaremos un código para procesar información, el diseño de la aplicación, la ejecución, 
así como la creación de una base de datos SQL, en la cual se almacenará la información capturada, y se 
conectará con la codificación hecha anteriormente; así mismo, crearemos un reporte que documentará el
proceso de realización de este proyecto, y una presentación.
También haremos pruebas, las cuales serán de retroalimentación para nuestra aplicación, estas pruebas también
serán registradas y entregadas.
